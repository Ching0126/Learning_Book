# C++ Qt 3个月实习导向学习计划
**核心目标**：掌握C++基础与Qt核心框架，完成3个递进式项目，具备Qt岗位实习能力。
**适用人群**：有C语言基础，对面向对象编程有初步认知（无Qt经验）。
**学习原则**：**理论+实战7:3配比**，每周任务必须落地代码，项目需完整提交到GitHub。
